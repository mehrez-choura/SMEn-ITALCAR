# SMEn-ITALCAR

This project contains an Energy Management System.

## How to Run

### Option 1: Open directly
Simply open `index.html` in your web browser.

### Option 2: Using Node.js
1. Ensure you have Node.js installed.
2. Run `npm start` in this directory.
3. Open the URL shown in the terminal (usually http://localhost:3000).

### Option 3: VS Code Live Server
1. Install the "Live Server" extension in VS Code.
2. Right-click `index.html` and select "Open with Live Server".

## Deployment 

This project is configured to automatically deploy to GitHub Pages when you push to the `main` branch.

### Setup Instructions
1. Go to your repository on GitHub.
2. Click on **Settings** > **Pages**.
3. Under **Build and deployment** > **Source**, select **GitHub Actions**.
4. The next time you push to `main`, the site will be deployed automatically.
